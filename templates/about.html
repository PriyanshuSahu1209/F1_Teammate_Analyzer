Teammate Analyzer

About This Project

This web app analyzes head-to-head race performance between a Formula 1 driver and all their teammates throughout their career. It uses historical F1 race data and summarizes which teammates performed most competitively, based on:

  • Percentage of races where the teammate finished ahead
  • Total points scored by each driver
  • Average finishing position across shared races

Technologies Used

  • Python + Flask – Backend and routing
  • Pandas + Plotly – Data analysis and visualizations
  • HTML + Bootstrap 5 – Responsive UI (based on Faster Theme)
  • CSV Dataset – From Kaggle (F1 historical race results)
